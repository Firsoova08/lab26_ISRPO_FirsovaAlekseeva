## Цель работы

Изучить и закрепить ключевые возможности современного JavaScript (ES6+):

- Деструктуризация
- Spread/Rest операторы
- Модули (Import/Export)
- Промисы и Async/Await
- Fetch API для работы с REST API
- Optional Chaining и Nullish Coalescing

---

## Основные выводы

В ходе выполнения лабораторной работы были изучены и применены на практике ключевые возможности современного JavaScript:

1. **Деструктуризация** значительно упрощает работу с объектами и массивами

2. **Spread/Rest операторы** делают код более чистым и читаемым

3. **Модули** позволяют организовать код в больших проектах

4. **Async/Await** упрощает работу с асинхронным кодом

5. **Fetch API** - современный стандарт для HTTP-запросов

6. **Optional Chaining** предотвращает ошибки при работе с вложенными объектами

Все эти концепции являются фундаментальными для современной веб-разработки и активно используются в React, Vue, Angular и других фреймворках.

## Реализованное приложение

В рамках лабораторной работы реализовано приложение **"Менеджер задач"** со следующим функционалом:

- Добавление новых задач
- Отметка о выполнении
- Удаление задач
- Фильтрация задач (все/активные/завершенные)
- Подсчет статистики
- Сохранение задач в localStorage

---

## Использованные концепции ES6+ в приложении

| Концепция | Применение |
|-----------|------------|
| Классы | `class TaskManager` |
| Стрелочные функции | Обработчики событий, методы массивов |
| Шаблонные строки | Создание HTML-разметки |
| Деструктуризация | Работа с объектами задач |
| Тернарный оператор | Условная отрисовка классов |
| Методы массивов | `push`, `find`, `filter`, `map`, `forEach` |
| Optional Chaining | `text?.trim()` |
| localStorage | Сохранение и загрузка задач |
| JSON | `JSON.stringify()`, `JSON.parse()` |
